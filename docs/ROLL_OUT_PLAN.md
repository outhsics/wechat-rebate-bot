# 完整落地方案（从0到上线）

## 阶段 1：跑通消息闭环（1天）
1. 配置公众号服务器地址到 `/wechat/callback`
2. 本地或服务器启动服务
3. 微信发消息验证 AI 回复与链接识别
4. 检查 `/api/link-logs` 是否有数据

验收标准：
- 公众号文本问答可用
- 三大平台链接可识别

## 阶段 2：接入真实联盟接口（2-4天）
1. 申请京东/拼多多/淘宝联盟开发者权限
2. 在 `app/services/affiliate/*.py` 替换 mock 实现
3. 增加接口失败重试与降级
4. 增加单元测试（真实响应样例）

验收标准：
- 返回真实券后价与佣金
- 每个平台至少 20 条样本验证通过

## 阶段 3：订单回传与结算（2-3天）
1. 接联盟订单回传（Webhook 或定时拉单）
2. 订单入库：pending -> paid -> settled
3. 结算池逻辑：达到门槛可返利
4. 管理台增加“确认打款”接口与日志

验收标准：
- 订单状态同步准确率 > 99%
- 可导出用户返利对账单

## 阶段 4：风控与合规（1-2天）
1. 加频控：单用户分钟级阈值
2. 加黑名单：恶意口令/域名
3. 菜单或自动回复加入免责声明
4. 隐私政策页上线（数据用途、保留时间）

验收标准：
- 高并发下系统稳定
- 用户投诉可追溯

## 阶段 5：运营优化（持续）
1. 增加关键词命令：`返利记录`、`提现`、`帮助`
2. 引导用户绑定手机号/支付宝
3. 监控：错误率、响应时长、订单转化率
